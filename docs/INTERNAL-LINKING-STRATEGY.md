## Link Equity Goals

1. **Home → Hub Pages**: Pass authority from homepage to /blog and /podcast hubs
2. **Hub → Individual**: Each hub distributes equity to individual articles/episodes
3. **Cross-Link Content**: Blog ↔ Podcast links create a content web
4. **Every Page → Pricing CTA**: All pages funnel to learn.lordsoflending.com/pricing
5. **Cornerstone Articles**: Future pillar content gets the most internal links

### Rule 4: Contextual Anchor Text (Not Generic)
- **Good:** "Learn more about [seller note structures](/seller-notes-deep-dive-lol-15) in our deep dive episode"
- **Bad:** "Click here for more" or "Read more"
- Anchor text should include the target keyword or topic naturally

### Rule 5: Hub Pages as Gatekeepers
- /blog and /podcast should always show the most recent and highest-value content first
- Add category/topic filters when article count exceeds 15
- Each hub page should include a prominent CTA to learn.lordsoflending.com/pricing

## Technical Notes

- Blog content lives in `/content/blog/{slug}.md` — cross-links go in the markdown
- Podcast show notes in `/content/podcast/{slug}.md` — same approach
- The `[slug]/page.tsx` template already handles markdown rendering with styled links
- Internal links use relative paths (`/slug`), external links use full URLs
- All links in markdown render with `target="_blank"` — **change internal links to open in same tab** (update `mdComponents.a` to check if href starts with `/`)
